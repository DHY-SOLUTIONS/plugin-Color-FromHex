# color_from_hex

Convert colors from HexColor

## Feature

- Color From Hex
- Color To Hex

## Getting started

How to use:

```dart
     Text( getColorFromHex('#FFFFFF').toString(),
          style: const TextStyle(fontSize: 20),), // Color(0xffffff)
```

```dart
    Text(getColorFromHex('#FFFFFF').toHex().toString(),
          style: const TextStyle(fontSize: 20)) //#ffffff
```
